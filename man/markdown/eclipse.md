% Eclipse

# Shortcuts

window > preferences > General > keys

(default)

- **Alt-F11**    : Fullscreen
- **Ctl-M**      : Maximize code window
- **Ctl-AvPag.** : next tab
- **Ctl-PrePag** : prev tab

(custom)

- **Ctl-b** : toggle bar

properties -> c++ general -> preprocessor -> providers -> CDT GCC Built-in Compiler Settings -> add "-std=c++14" to command input box
